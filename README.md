# DUNGSIL's Agent Skills

개인 취향과 경험을 기반으로 한 [에이전트 스킬](https://agentskills.io/home) 모음집

## 설치
```
pnpm dlx skills add dungsil/skills -g
```

## 개발/검증

```bash
bun install
bun run validate:skills
```
